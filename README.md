# ScratchPenalties

Controls:

---------------
General
---------------
- R - Reset
- M - Pause Menu

---------------
Local & vs COM PK
---------------
- Q - Top left
- W - Top middle
- E - Top right
- A - Bottom left
- S - Bottom middle
- D - Bottom right

---------------
Local GK
---------------
- DOWN ARROW - Top left
- NONE - Top middle
- UP ARROW - Top right
- LEFT ARROW - Bottom left
- NONE - Bottom middle
- RIGHT ARROW - Bottom right
